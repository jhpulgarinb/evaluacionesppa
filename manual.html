Uso del Formulario

  • **Captura de Datos**: Rellene cada campo del formulario. La informacion es crucial para el analisis posterior.
  • **Guardar Registros**: Presione el boton **"Guardar"** despues de cada evaluacion. Esto almacenara los datos en la memoria interna de su dispositivo.
  • **Visualizar Datos**: Use el boton **"Ver Registros"** para revisar los datos que ha guardado localmente.
  • **Exportar a Excel**: Genere una copia de seguridad en formato Excel en cualquier momento para su revision o uso externo.

Sincronizacion de Datos

El envio de datos a la base de datos en linea es un paso fundamental para centralizar la informacion.

  • **Enviar a la Nube**: Cuando tenga una conexion a internet estable, presione el boton **"Enviar a Base de Datos"**. Esto subira todos los registros guardados a la Base de Datos externa.
  • **Sincronizacion**: Al enviar los registros, se eliminaran de la memoria del dispositivo para evitar duplicados y liberar espacio.
