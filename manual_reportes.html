Manual del Sistema de Insumos y Labores

Guia de procedimientos para el registro de insumos y labores diarias.

Procedimiento de uso

Este manual describe el flujo de trabajo para el registro de insumos y labores diarias. Se debe seguir el proceso en el orden indicado para garantizar la precision de los datos.

Registro de Entrega de Insumos

Este es el primer paso del proceso. Se utiliza cuando se entrega un insumo a un operario.

  • Seleciona Entrega de Insumos.
  • Completa los campos de Informacion General, como la Fecha, la Finca, el Responsable, el Codigo de Operario y el Insumo que se entrega.
  • En la seccion de Cantidades, ingresa la Cantidad Entregada al operario.
  • El sistema calculara el Consumo Total, y debes registrar la Labor a Realizar y cualquier Observacion pertinente.
  • Haz clic en el boton "Guardar" para almacenar el registro localmente, o en "Enviar a Base de Datos" para enviarlo a la base de datos externa.

Registro de Devolucion de Insumos

Este paso es opcional y se realiza si un operario devuelve una porcion del insumo que no utilizo.

  • Selecciona Devolucion de Insumos.
  • En la seccion "Buscar Registro de Entrega", ingresa el Codigo de Operario y el Insumo que se esta devolviendo.
  • Haz clic en el boton "Buscar Registro" para encontrar el registro de entrega inicial.
  • Una vez que se muestre la informacion de la entrega, ingresa la Cantidad a Devolver en el campo correspondiente.
  • El sistema recalculara automaticamente el Consumo Actualizado.
  • Haz clic en "Guardar Devolucion" para actualizar el registro inicial y reflejar el consumo real del insumo.

Reporte de Labores Diarias

Este es el paso final para registrar el avance del trabajo del operario.

  • Selecciona Reporte de Labores.
  • En la seccion de Datos del Operario y la Labor, ingresa el Codigo de Operario y la Labor que va a reportar.
  • Puedes elegir entre "Nuevo Reporte" o "Buscar y Editar" un reporte existente.
  • Completa la informacion detallada de la labor, incluyendo la Fecha, la Finca, el Responsable, las Palmas Visitadas, la Cantidad Realizada y el area Recorrida.
  • Utiliza la seccion de Centros de Costo para agregar el detalle de la labor por cada centro de costo, si aplica.
  • Haz clic en "Guardar" para almacenar los datos del reporte. Puedes ver un resumen o exportar el reporte a Excel.
